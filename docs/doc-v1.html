AI Agent CLI System

Quản lý AI agents, skills, và workflows từ command line. AI agents có thể tự đọc docs và vận hành toàn bộ hệ thống.

🚀 Bắt đầu 📖 Kiến trúc
KIẾN TRÚC

6 Hệ Thống Con

ZhiYing được xây dựng từ 6 subsystems hoạt động cùng nhau

🤖

Agent Manager

Tạo và quản lý AI agents với persona, routine, skills, và cloud API keys. Lưu trữ JSON, CRUD qua CLI hoặc API.

⚡

Skill System

Skills = workflow templates có sẵn. 4 skills mặc định. Agents được gán skills để thực hiện tác vụ.

🔄

Workflow Engine

DAG executor sử dụng Kahn's algorithm (topological sort). Nodes kết nối qua ports, hỗ trợ loop iteration.

🌐

API Server

FastAPI REST server với Swagger docs. CRUD endpoints cho agents, skills, workflows. Auto-gen API documentation.

🖥️

WebUI Dashboard

Giao diện đồ họa tích hợp sẵn. Quản lý Agents, Browser Profiles, và xây dựng Workflow bằng thao tác kéo-thả (Drag & Drop) trực quan.

📖

AI Documentation (.agents/skills/SKILL.md)

Tài liệu viết dưới dạng SKILL.md để AI agents (Gemini, Claude, GPT) có thể đọc và tự vận hành toàn bộ hệ thống mà không cần hướng dẫn của người dùng. Bao gồm CLI reference, API endpoints, node types, workflow JSON schema, và common patterns.

NODE TYPES

7 Loại Node

Mỗi node có typed ports (inputs/outputs) kết nối thành workflow DAG

📝 Text Input text_input
Cung cấp text tĩnh làm đầu vào cho workflow.
Outputs: content, lines
🔄 Loop loop
Lặp qua danh sách items, thực thi downstream mỗi iteration.
In: items → Out: current_item, index
🐍 Python Code python_code
Chạy code Python tùy ý. Access inputs qua biến.
In: text_input, json_input → Out: result
🌐 API Request api_request
HTTP requests (GET/POST/PUT/DELETE) tới APIs.
In: url, body → Out: response, status_code
💻 Run Command run_command
Chạy shell commands, capture stdout và exit code.
In: command → Out: stdout, exit_code
🧠 AI Inference ai_node
Gọi AI model (Ollama local hoặc Cloud API).
In: prompt, context → Out: response
📤 Output output
Hiển thị hoặc lưu kết quả cuối cùng ra file.
In: data → Out: file_path
