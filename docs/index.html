AI Agent CLI System

Hệ thống lõi mở rộng quản lý AI Agents, Workflow Builder, Teams 3D Studio, Story Engine và Browser Automation. Tự động hóa tác vụ với giao diện web thời gian thực.

KIẾN TRÚC

10 Hệ Thống Con

ZhiYing được xây dựng từ 10 subsystems hoạt động cùng nhau

🤖

Agent Manager

Tạo và quản lý AI agents với persona, routine, skills, và cloud API keys. Lưu trữ JSON, CRUD qua CLI hoặc API.

⚡

Skill System

Skills = workflow templates có sẵn. 4 skills mặc định. Agents được gán skills để thực hiện tác vụ.

🔄

Workflow Engine

DAG executor sử dụng Kahn's algorithm (topological sort). Nodes kết nối qua ports, hỗ trợ loop iteration.

🌐

API Server

FastAPI REST server với Swagger docs. CRUD endpoints cho agents, skills, workflows. Auto-gen API documentation.

🔌

Extension System

Kiến trúc extension mở rộng: Browser, Web Dashboard, Marketplace. Enable/disable qua CLI, auto-inject commands và API routes.

🌐

Browser Engine

Quản lý browser profiles, proxy, fingerprint, cookies. Auto-login Google với TOTP 2FA. AI automation qua prompts. Profile settings với proxy, fingerprint OS/device, Google Account.

🎨

Web Dashboard

Giao diện web SPA tại localhost:5295/dashboard. Quản lý agents, browser profiles (với profile settings ⚙️), workflows, skills, marketplace và settings. Auto-polling trạng thái browser realtime.

👥

Teams Agents

Multi-agent orchestration: tạo team từ template, org chart phân cấp, giao task tự động cho nhiều agents theo strategy (sequential, parallel, hierarchical).

🏢

3D Studio

Văn phòng 3D isometric (Three.js) với nhân vật Roblox-style. 15+ asset types (bàn, ghế, đèn...). Scene management cho từng team. Fullscreen mode.

🎬

Story Engine

Kể chuyện 3D với Script Editor tự động sinh hội thoại. Phát kịch bản trực quan qua 3D Story Player với bong bóng thoại (Speech Bubbles).

📖

AI Documentation (.agents)

Tài liệu SKILL.md để External AI (Gemini, GPT) đọc hiểu kiến trúc và tự vận hành, viết code plugin mở rộng cho hệ thống không cần con người.

NODE TYPES

7 Loại Node

Mỗi node có typed ports (inputs/outputs) kết nối thành workflow DAG

📝 Text Input text_input
Cung cấp text tĩnh làm đầu vào cho workflow.
Outputs: content, lines
🔄 Loop loop
Lặp qua danh sách items, thực thi downstream mỗi iteration.
In: items → Out: current_item, index
🐍 Python Code python_code
Chạy code Python tùy ý. Access inputs qua biến.
In: text_input, json_input → Out: result
🌐 API Request api_request
HTTP requests (GET/POST/PUT/DELETE) tới APIs.
In: url, body → Out: response, status_code
💻 Run Command run_command
Chạy shell commands, capture stdout và exit code.
In: command → Out: stdout, exit_code
🧠 AI Inference ai_node
Gọi AI model (Ollama local hoặc Cloud API).
In: prompt, context → Out: response
📤 Output output
Hiển thị hoặc lưu kết quả cuối cùng ra file.
In: data → Out: file_path

❯ zhiying browser status

⏹ Browser 'my-profile' stopped.

# ═══ Profile Settings (⚙️ trên Dashboard) ═══
# Mỗi profile có thể cấu hình:
• Proxy: socks5://user:pass@host:port
• Fingerprint: OS (Windows/macOS) + Browser
• Google Account: email|password|recovery|2fa_secret

# Auto-Login Flow khi mở browser:
1. Đọc google_account từ profile config
2. Kiểm tra đã login (myaccount.google.com)
3. Nếu chưa → tự nhập email/password
4. 2FA → gọi API /api/v1/browser/2fa (TOTP live code)
5. Login xong → tiếp tục lệnh ban đầu

TEAMS

Teams — Multi-Agent Orchestration

Tạo đội nhóm AI agents, phân vai trò, giao task tự động theo chiến lược

👥

Team Builder

Tạo team từ template có sẵn hoặc custom. Mỗi team có roles (vai trò), agents được gán vào roles, và strategy thực thi (sequential, parallel, hierarchical).

📊

Org Chart & Multi-seat

Sơ đồ tổ chức phân cấp kéo thả. Hỗ trợ multi-seat assignment: 1 bàn họp lớn có thể gán cho nhiều agent cùng làm việc với thuật toán đối diện thông minh.

🚀

Task Delegation

Giao task cho toàn bộ team — hệ thống tự phân công cho từng agent theo role và strategy. Xem kết quả từng agent realtime.

📋

Templates

Chọn template có sẵn (Dev Team, Marketing, Support...) với roles đã định nghĩa sẵn. Chỉ cần gán agents và bắt đầu.

Teams API
# Tạo team từ template
❯ POST /api/v1/multi-agents/teams/from-template
{ "template_id": "dev_team", "name": "My Dev Team", "agent_assignments": {...} }

# Delegate task cho team
❯ POST /api/v1/multi-agents/teams/{id}/delegate
{ "task": "Viết bài marketing cho sản phẩm mới" }
✅ 3 agents hoàn thành: Researcher → Writer → Editor

# Xem org chart & gán agents
📊 CEO (🧠 GPT-4) → 📋 Manager (🤖 Qwen) → 💻 Developer (⚡ Gemini)
